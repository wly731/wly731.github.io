Ruby Enumerable methods

One of the most basic enuemrable methods to loop through and change the content of your array or hash

31st January 2016

You may have already heard about looping. As a newbie the loop that you feel most comfortable with is probably while loop with a loop counter "i". However, if you want to iterate through an array or hash, there is actually a better way to do so. In this blog post, I will introduce you an enumerable method - Enumerable#map


What does #map do?

#map is actually the same as #collect. What they do is to iterate through every element of the data type, run the code block on each of them, then return the new element. (or replace the original element with the new element if the destructive method is called, which is #map!) It is like remapping the original object through some transformation.


How to use #map?

Here are a few examples:

num = [1,2,3,4]

num.map{ |number| number*2 } #=> [2,4,6,8]


name = ["victor","wong"]

name.map{ |name| name.capitalize } #=> ["Victor","Wong"]
