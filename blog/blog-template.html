Version Control System, aka VCS, is a word that you should know as a beginner in this field. Actually this is no new thing, as most of you may probably have used Google Doc before, in which different people can work on the same document remotely and simultaneously, and the changes are tracked accordingly. Well that is basically what a VCS is about: it allows people to collaborate, track changes along with who made the change and why they made it, and also shift between different versions of work. The benefit of VCS includes increasing working efficiency, code due diligence, working clairity, improving division of labor, etc. Especially in an environment where team collabartion is of utmost importance.

Here I am going to introduce a very nice and easy to use VCS, which is called Git. You can easily install it by following the instruction here: https://confluence.atlassian.com/bitbucket/set-up-git-744723531.html So how does git actually help you track changes? Before I walk into that I would like to introduce some common terminology in git. Firstly, "git add" means adding a file to be tracked. Secondly, branch means version of work, you can create multiple branches and work on them. Thirdly, commit means save. Every time you push the changes to the master code, you have to first commit (save) and then push. Git helps you keep track of changes by allowing you to create multiple branches and add different files to be tracked. After each change you commit and leave a commit message. User can then check the changes history by typing "git log"

You may have also heard of GitHub and wondered is it the same is Git. Well actually they are not the same. Git is installed and used locally in your computer, while GitHub is a cloud platform that allows different users to upload and merge their work remotely. You can think of GitHub as the Google Drive/Dropbox of sharing the your local git. Why use GitHub to store your code? The answer is simple and similar to why you use Dropbox to store your work - you can easily access your work anywhere and share your work to anybody, with the help of network!
